## Описание предметной области

**Шифр Билайна** — это метод перестановочного шифрования, при котором текст записывается зигзагом по нескольким строкам, а затем считывается построчно. Реализация с динамическим числом строк (ключом) позволяет менять уровень стойкости шифра&

### Характеристики шифра:
- **Динамическое число строк** (настраивается пользователем от 2 до 10)
- **Чтение зигзагом** — движение по строкам вниз-вверх
- **Сохранение порядка символов** в каждой строке
- **Полная обратимость** — шифрование и дешифрование взаимно обратны

### Область применения:
- Образовательные цели (демонстрация перестановочных шифров)
- Шифрование коротких текстовых сообщений
- Изучение основ криптографии

## Диаграмма вариантов использования
```
@startuml
left to right direction
skinparam packageStyle rectangle

actor "Пользователь" as User

rectangle "Шифр Билайна (Двухлинейный)" {
  
  usecase "Шифрование текста" as ShText
  usecase "Дешифрование текста" as DeText
  usecase "Настройка количества строк" as Quan
  usecase "Ввод исходного текста" as VText
  usecase "Просмотр результата" as Result
  usecase "Валидация входных данных" as Val
  usecase "Визуализация процесса заполнения" as Vis
  usecase "Обработка ошибок" as Err

  User --> ShText
  User --> DeText
  User --> Quan
  User --> VText
  User --> Result
  
  ShText ...> Val : <<include>>
  DeText ...> Val : <<include>>
  ShText ...> Vis : <<extend>>
  Quan ...> Val : <<include>>
  VText ...> Val : <<include>>
  
  Err <... ShText : <<extend>>
  Err <... DeText : <<extend>>
}

@enduml
```
